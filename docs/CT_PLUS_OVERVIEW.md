# CT+ 캠페인 리포트 — 기능 현황 및 로드맵

> 최종 수정: 2026-04-02

---

## 1. 주요 운영 기능

### 1-1. 리포트 CSV 데이터 입력
- **모티브 광고성과 수집기**에서 수집한 매체별 로우 데이터(CSV)를 시스템에 직접 연동
- 매체(네이버 GFA / 카카오모먼트 / Google / META) 통합 파싱 후 성과 분석 기반 데이터로 활용

### 1-2. DMP 정산 데이터 확인
- 캠페인 설정(매체비 · DMP 수수료 · 대행사 수수료)을 기반으로 정산 금액 자동 산출
- 매체별 · DMP별 · 대행사별 세부 정산 내역 조회 및 스냅샷 저장

### 1-3. 목업 이미지 생성
- 업로드된 광고 소재를 기반으로 시각적 목업 이미지 생성
- 광고 소재 규격 검수 및 보고서 첨부용 활용

---

## 2. 당면 과제 및 개선 계획

### 2-1. CSV ↔ 광고 계정 매칭 자동화
| 현황 | 목표 |
|------|------|
| 수집된 CSV와 실제 광고 계정을 수동으로 연결 | 캠페인명 기반 자동 매칭 + 수동 오버라이드 병행 |
| 휴먼 에러 발생 가능 | 퍼지 매칭 알고리즘으로 오류 최소화, 입력 시간 단축 |

### 2-2. 목업 이미지 AI 합성 고도화
| 현황 | 목표 |
|------|------|
| 단순 소재 목업 생성 | AI 기반 지면 합성 (게재 확인 목업) |
| 지면과 소재의 별도 분리 출력 | 카카오 비즈보드 · 네이티브 등 실제 지면에 소재를 자연스럽게 합성 |

- 활용 모델: `gpt-image-1` (OpenAI Images API)
- 지원 지면: 비즈보드 / 네이티브 1200×600 / 네이티브 1000×800 / 카카오 View

### 2-3. DMP 정산 편의성 개선
- DMP 캠페인별 · 계정별 정산 내역 통합 조회
- CSV 실적 데이터와 등록 캠페인 정산 기준 간 대사(對査) 기능 강화

---

## 3. 로드맵 — CT / CTV / CT+ 통합 정산 플랫폼

```
현재                          목표
─────────────────────         ──────────────────────────────────────
CT+  → 본 시스템 (운영 중)    ┐
                               ├─ 단일 통합 정산 관리 플랫폼
CT   → 플랫폼 현황/관리 시트  ┤    · CT / CTV / CT+ 전 라인
CTV  → 플랫폼 현황/관리 시트  ┘    · 매체비 · DMP · 대행사 수수료 일원화
```

### 통합 시 핵심 목표

| 항목 | 내용 |
|------|------|
| 데이터 일원화 | MongoDB 기반 캠페인 · 대행사 · 광고주 · 운영자 마스터 데이터 공유 |
| 정산 자동화 | CSV 업로드 → 매칭 → 수수료 계산 → 리포트 생성 전 과정 자동화 |
| 리포트 고도화 | 영상 성과(VTR/CPV) · 캠페인별 · 계정별 브레이크다운 포함 HTML 리포트 |
| 확장성 | CT / CTV 라인 데이터 스키마 통합, 향후 신규 매체 추가 대응 |
